# 💰 defi-dex-interface

Uniswap-style DEX frontend with swap, pool and limit order features

## Features

- Token swaps
- Liquidity pools
- Limit orders
- Price charts
- Slippage protection

## Tech Stack

- **Smart Contracts:** Solidity ^0.8.20
- **Framework:** Hardhat
- **Frontend:** Next.js + TypeScript
- **Web3:** ethers.js / wagmi

## Getting Started

```bash
npm install
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.js --network localhost
```
